## Description

Este es un projecto de prueba con el curso de Udemy de Fernando Herrera

## Installation

```bash
$ npm install
$ docker-compose up -d
```

## Librerias Usadas

  - npm i @nestjs/config
  - npm install --save @nestjs/typeorm typeorm pg
  - npm i joi
  - npm i dotenv
  - npm i class-validator class-transformer
  - npm i -D @types/multer
  - npm install --save @nestjs/passport passport 
  - npm install --save @nestjs/jwt passport-jwt
  - npm install --save-dev @types/passport-jwt

## Todos los pasos

1 - Primero creo el docker compose para montar la base de datos
2 - Luego isntalo el paquete para leer variables de entorno, y lo importo en App.Module
